This repisitory contains all the code and data files for our depression project. We have attached the raw CSV file of reddit posts, the embedding file, the embedding code,
and all the required code for our chrome extension.

To make a functioning chrome extension, follow steps listed below:
1) Upload the embedding csv to a gdrive folder and mount the gdrive in the backendCode.py file attached under extension-files
2) Go to openAI and buy a API key. ("company" aspect here: we "partner" with openAI and get money from their API revenue). Paste into the backendCode.py file
3) Download the depr-files code and open up chrome-extension management
4) Upload the depr-files folder and pin the extension (labelled H in white and gray background)
5) Run the backendCode.py file in gcolab and then take the ngrok link to paste into chatbot url

PRIVACY WARNING: Tracks keystrokes. We do not use these for any privacy breaching purposes. If you disable the code, the extension will still run so...
If you type any depression related messages, the extension WILL pop-up and will not let you tab switch until you confirm ok.
